====== 03 引用调用 ======
Created Monday 10 June 2019

C++引用调用			该方法把参数的引用复制给形式参数。在函数内，该引用用于访问调用中要用到的实际参数。这意味着，修改形式参数会影响实际参数。

向函数传递参数的引用调用方法，把引用的地址复制给形式参数。在函数内，该引用用于访问调用中要用到的实际参数。这意味着，修改形式参数会影响实际参数。

按引用传递值，参数引用被传递给函数，就像传递其他值给函数一样。你需要声明函数参数为引用类型。

这里利用了数异或的性质，一个数与其本身异或等于0,0与一个数异或不改变该数。

int swap(int &a, int &b)
{
	int temp;
	temp = a ^ b;
	a = temp ^ a;
	b = temp ^ b;
	return 0;
}
