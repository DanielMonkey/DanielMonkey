====== 03 Traversal with a for loop ======
Created Tuesday 09 July 2019

A lot of computations involve processing a string one character at a time. Often they start at the beginning, select each character in turn, do something to it, and continue until the end. This pattern of processing is called traversal.

One way to write travesal is with a while loop

Another way to write a traversal is with a for loop
