====== 1.3 What is debugging ======
Created Wednesday 20 March 2019

Programming is error-prone.
For whimsical reasons, programming errors are called bugs and the process of tracking them down is called debugging.

Three kinds of errors can occur in a program: syntax errors, runtime errors, and sematic erros.
It is useful to distinguish between them in order to track them down more quickly.
